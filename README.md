O objetivo desta atividade foi sistematizar tarefas do cotidiano. A proposta é transformar ações automáticas do dia a dia em processos lógicos passo a passo, garantindo que nenhuma etapa essencial seja esquecida.

🎯 Objetivos do Projeto
Organizar situações do cotidiano para resolver problemas complexos de forma fragmentada.

Estruturar o passo a passo lógico para fazer atividades do cotidiano.

Utilizar word para criar fluxogramas e pseudocódigos.

📋 Cenários Mapeados
Para usar a lógica, desenvolvi algoritmos para os seguintes cenários:

🚗 Trocar um pneu

💡 Trocar uma lâmpada.

🚌 Pegar um ônibus.

🍰 Fazer um bolo.

💼 Ir ao trabalho.

🛠️ Ferramentas e Metodologia
A atividade foi feita utilizando o Word, onde foram feitos:

Pseudocódigo: Uma forma simples de escrever um algoritmo, para facilitar o entendimento da lógica antes do código real. 

Fluxograma: Uma representação gráfica que utiliza símbolos padronizados para ilustrar os passos a passos.
